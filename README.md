# Система оптимизации транспортной логистики

Этот репозиторий содержит описание и рабочий прототип системы управления цепочками поставок.
Он включает алгоритм выбора оптимального маршрута доставки с учетом географического расположения объектов,
дорожных условий и загруженности транспорта.

## Быстрый старт

1. Создайте виртуальное окружение и установите зависимости (дополнительно можно поставить `PyYAML` для поддержки YAML-файлов).
2. Запустите оптимизацию маршрута на примерном сценарии:
   ```bash
   python -m supply_chain.app examples/sample_network.json --start mine --goal plant --departure 480
   ```
3. Для формата JSON добавьте `--format json`, для включения детализированных логов `--log-level DEBUG`.
4. Выполните тесты:
   ```bash
   python -m pytest
   ```

## Структура

- `supply_chain/` — исходный код приложения.
- `examples/sample_network.json` — пример описания транспортной сети с профилями затрат.
- `docs/supply_chain_routing.md` — подробная документация по архитектуре и алгоритму.

## Ключевые возможности

- Валидация конфигураций сценариев с понятными ошибками при нарушении требований.
- Расширенный CLI с логированием и выводом результатов в текстовом и JSON форматах.
- Поддержка профилей стоимости во времени для топлива, рисков и потерь.
- Автотесты, проверяющие корректность маршрутизации и работы валидации.
